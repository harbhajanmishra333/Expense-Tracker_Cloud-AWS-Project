DEPLOYMENT STEPS (Quick Version)
================================================================================

1. Install dependencies:
   npm install

2. Deploy to AWS:
   serverless deploy

3. Copy output values:
   - ApiEndpoint
   - UserPoolId
   - UserPoolClientId

4. Configure frontend:
   cp frontend/config.template.js frontend/config.js
   (Edit with your values)

5. Run frontend:
   cd frontend
   python -m http.server 8000

6. Open browser:
   http://localhost:8000
